Privacy Policy — Histogram Pro

Last updated: June 2026

1. Overview

Histogram Pro is a Power BI custom visual developed by TCViz (Tino Callarisa). This policy describes how data is handled when you use this visual.

2. Data Collection

Histogram Pro does not collect, transmit, or store any data. All computation happens locally inside Power BI using your report data. No data is ever sent to external servers.

3. External Connections

The visual makes no external network requests. It operates entirely within the Power BI rendering environment. The only external API used is Microsoft's own IVisualLicenseManager to verify AppSource license status — this is a standard Microsoft API and no user data is involved.

4. Cookies

This visual does not use cookies or any browser storage.

5. Third-Party Services

No third-party analytics, tracking, or advertising services are used.
